# Messaging App Comparison

An interactive React app for comparing secure messaging apps across key privacy, security, and usability dimensions.

## What it covers

Compares apps including Signal, Matrix/Element, Session, and others across:

- **Encryption** — protocol, E2E, Perfect Forward Secrecy, post-compromise security, quantum resistance
- **Architecture** — centralized, federated, decentralized, P2P
- **Privacy** — metadata protection, anonymous signup, phone number requirement, jurisdiction
- **Usability** — group size limits, voice/video, multi-device, platforms
- **Operations** — self-hostable, rate limiting behavior, audit status, user base

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
